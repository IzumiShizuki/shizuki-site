# 术语总表

## 1. 核心术语

| 规范术语 | 英文 | 代码侧命名 | 禁止继续混用的别名 | 说明 |
| --- | --- | --- | --- | --- |
| 轻应用 | LightApp | `LightApp*` | `widget`、`app window`、`小工具` | 用户主动打开的工具型功能 |
| 小组件 | Widget | `*Widget` | `轻应用` | 仅指首页嵌入块或固定展示块 |
| 悬浮栏 | Floating Dock | `dock` | `right rail`、`floating rail` | 包含悬浮球、槽位、集合、扩展面板的整体入口系统 |
| 悬浮球 | Floating Ball | `ball` | `menu button`、`drop ball` | 悬浮栏的圆形触发器 |
| 扩展面板 | Dock Panel | `dockPanel*` / `panelKind` | `picker-panel`、`submenu-panel`、`panelMode` | 悬浮球展开后用于显示可选项或集合内容的面板 |
| 集合 | Collection | `collection*` | `folder`、`group` | 悬浮栏中的分组容器 |
| 轨道槽位 | Rail Slot | `rail_slots` / `slot*` | `ball slot`、`floating slot` | 悬浮栏中的单个可分配槽位 |
| 页面 | Page | `*Page` | `view`、`screen` | 路由级组件 |
| 子视图 | View | `*View` | `page` | 页面内部的主要业务视图 |
| 轻应用窗体 | Window | `*Window` | `panel`、`page-mode` | 可拖拽、可复用的轻应用承载壳 |
| 弹窗 | Dialog | `*Dialog` | `modal`、`popup` | 模态层 |
| 底部层 | Sheet | `*Sheet` | `mobile-dialog` | 移动端底部滑出层 |
| 短暂浮层 | Popover | `*Popover` | `bubble` | 锚点附近的轻量信息层 |
| 布局骨架 | Layout / Shell / Stage | `*Layout` / `*Shell` / `*Stage` | `container`、`wrapper` | 负责排版和区域组织，不承载业务状态 |
| 导航轨道 | Rail | `*Rail` | `dot sidebar` | 点状或轨道式导航容器 |

## 2. 本仓库关键落点

- `fronted/vue3-merged/src/components/LevitationBall.vue`
  - `float-ball` = 悬浮球
  - `dock panel` = 悬浮球展开面板
- `fronted/vue3-merged/src/utils/lightAppsState.js`
  - `rail_slots` = 主持久化结构
  - `collections` = 悬浮栏集合定义
  - `ball_slots` = 历史兼容别名
- `fronted/vue3-merged/src/components/lightapps/*Window.vue`
  - `*Window` = 轻应用窗体
- `fronted/vue3-merged/src/components/apps/*Widget.vue`
  - `*Widget` = 首页嵌入块，不等于 `LightApp`

## 3. 对外说明口径

- 面向用户或产品文档时，优先使用中文规范术语。
- 面向代码和接口时，保留英文标识与字段原文。
- 一个术语只能有一个主名称；别名仅允许在遗留兼容台账中出现。
