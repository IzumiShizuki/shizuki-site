# 

开场白，假期当中可能有冷淡(我)可能有





弟兄姊妹，今天是寒假刚刚结束后的第一个主日。这个时间点其实很特别。因为一个假期过去，人的属灵生命常常会被显明出来。平时在规律的聚会里，在弟兄姊妹的陪伴里，在教会生活的节奏里，我们好像还觉得自己属灵情况不错；但一旦离开了日常的交通，离开了稳定的聚会，离开了彼此的守望，很多人就会发现，自己其实并没有想象中那么刚强。

我自己在假期里就很深地经历这一点。当我缺少日常生活中和弟兄姊妹的交流时，我发现自己很容易变得软弱。不是一下子跌倒成很大的样子，而是慢慢地冷淡下来。圣经不太想读了，祷告也容易敷衍了，各样该殷勤的事都开始懈怠。表面上可能只是生活节奏变了，但实际上，里面的属灵状态已经在下滑。

这个经历让我重新明白一件事：**神从来没有把基督徒设计成独行的人。**
 神救我们，不只是把我们一个个从世界里救出来；神也把我们带进一个家，带进一个身体，带进一个彼此相顾、彼此建立的群体里。这个群体，就是教会。

所以，今天我们要借着希伯来书10章24到25节，一起来思想一个很重要的题目：**在假期之后，我们怎样重新回到教会的连结中，怎样真实地融入神的家。**

经文说：“又要彼此相顾，激发爱心，勉励行善。你们不可停止聚会，好像那些停止惯了的人，倒要彼此劝勉；既知道那日子临近，就更当如此。”

这段经文不是单单告诉我们“要来聚会”，更是在告诉我们，基督徒的生命若要健康，就不能脱离群体。因为神不是要我们独自坚持，而是要我们在彼此守望中一起往前走。

首先我们来看第一点：**离开属灵群体，人很容易变得软弱冷淡。**

很多时候，我们低估了孤立对灵命的影响。我们以为，只要我自己信得够认真，自己安排一下时间，自己也能祷告、读经、爱主。但是现实常常不是这样。人一旦离开属灵群体，就容易松下来；一松下来，就容易冷淡；一冷淡，就会开始给自己很多理由；再往后，甚至会慢慢觉得，不聚会也没有什么，不交通也没有什么，不追求也没有什么。

圣经里有一个很典型的例子，就是多马。主耶稣复活以后，门徒聚在一起，主向他们显现。可是约翰福音20章告诉我们，那一次多马不在。其他门徒都看见主了，经历了主复活的大能，也得着了喜乐和平安；只有多马还留在怀疑里。他甚至说：“我非看见……我总不信。”

弟兄姊妹，请注意，多马的问题，并不只是因为他性格比较理性，也不只是因为他信心比较小；还有一个很现实的原因，就是在门徒一同聚集、主向他们显现的时候，他缺席了。他错过了那个群体一起领受恩典的时刻。

这就是属灵生活里一个很真实的原则：**当我们离开弟兄姊妹，离开教会生活，离开属灵群体时，我们就更容易留在自己的感觉里、自己的判断里、自己的灰心里。**
 没有人提醒我，没有人和我一起祷告，没有人问我最近怎么样，没有人陪我走，那我就很容易在里面越走越弱。

所以希伯来书说，不可停止聚会。因为停止聚会不是一个中性的动作，它常常意味着属灵生命开始脱节。不是说一个人偶尔因特殊原因缺席就等于不属灵，而是说，当人渐渐习惯了离开群体、游离在外，他的心就很容易慢慢冷下来。

也许假期已经让我们看见这件事了。也许有些弟兄姊妹这个寒假里属灵状态并不好。读经不规律，祷告很散，心容易被手机、娱乐、懒散、世俗的节奏拖住。那今天神借着这篇道不是来定罪我们，而是要提醒我们：**你之所以会软弱，不是因为你比别人差，而是因为你本来就不是被设计成独自争战的人。**

传道书说，两个人总比一个人好；若是跌倒，这人可以扶起他的同伴。属灵的路也是一样。很多时候，神扶持我们，不是直接从天上掉下一股力量，而是借着教会中的弟兄姊妹，借着一句问候、一次代祷、一场交通、一同敬拜，把我们重新托起来。

接着我们来看第二点：**教会不是可有可无的聚会点，而是基督的身体、我们的家。**

为什么我们需要融入教会？因为教会不是一个宗教场所，而是一个生命共同体。哥林多前书12章告诉我们，教会是基督的身体，我们每一个信主的人都是这身体上的肢体。既然是身体，就表示没有一个肢体是独立存在的。手不能说我不需要脚，眼不能说我不需要耳；同样，一个基督徒也不能说，我只要自己和主好就行了，我不需要教会。

今天很多人对教会的理解，是“我去不去”“我喜不喜欢”“适不适合我”“有没有满足我的需要”。但圣经对教会的理解不是消费式的，而是身体式的。不是“这间教会给了我什么”，而是“神把我放在这个身体里，要我怎样和众圣徒一起被建造，也一起建造别人”。

使徒行传第二章让我们看见，初代教会信主以后，不是各自回家单独做基督徒，而是恒心遵守使徒的教训，彼此交接，擘饼，祈祷。他们在一起敬拜，在一起生活，在一起经历神的恩典。换句话说，教会从一开始就不是一种“偶尔参与”的活动，而是一种“共同生活”的实际。

所以，融入教会，不是让我们变成热闹一点而已；融入教会，是神保护我们、塑造我们、使用我们的方式。你在教会里，不只是来听道；你在教会里，也是被爱、被提醒、被扶持、被修剪、被装备。与此同时，你也成为别人的帮助、别人的安慰、别人的鼓励、别人的祝福。

有时候我们觉得自己不重要，觉得“少我一个也没关系”。但哥林多前书清楚告诉我们，身体上的肢体没有一个是多余的。也许你不是站讲台的人，也许你不是特别显眼的人，但你在教会中的存在，依然有神所定的价值。一个眼神的关心，一句诚实的问候，一次安静的代祷，一个忠心的服事，都可能成为神建立教会的器皿。

所以，教会不是一个你有空才来的地方，而是你属灵生命得以长成的地方；教会不是一个你来“打卡”的地方，而是你真正所属的家。

最后我们来看第三点：**真正融入教会，就是彼此相顾、彼此委身、彼此服事。**

希伯来书10章24节说：“又要彼此相顾，激发爱心，勉励行善。”这里的重点不是“各自努力”，而是“彼此相顾”。一个真正融入教会的人，不只是自己来聚会而已，他开始在意别人。他会想：最近某某弟兄怎么没来了？某某姐妹最近是不是遇到什么难处？我有没有可以陪伴的人？有没有我可以参与的服事？有没有我可以承担的责任？

换句话说，融入教会不是“我出现在这里”，而是“我把自己交在这个身体里”。
 不是只做礼拜，而是进入关系；不是只听信息，而是愿意被牧养，也愿意去关心别人；不是只做旁观者，而是开始参与建造。

一个真正融入教会的人，至少会有三个表现。

第一，他会**稳定地回到聚会生活里**。
 因为他知道，聚会不是形式，而是生命的需要。敬拜、听道、祷告、团契，这些不是宗教项目，而是神供应我们生命的管道。

第二，他会**主动进入真实的关系**。
 很多人长期在教会里，却并没有真正融入，因为他只停留在表面的来往里。他来了、听了、散了，但没有让别人真正认识他，也没有真正去认识别人。可是教会生活若没有真实关系，就很难有真实建造。我们需要学习敞开，学习交通，学习让别人知道我们的软弱，也学习陪伴别人的挣扎。

第三，他会**在服事中找到自己的位置**。
 以弗所书4章说，全身都靠基督联络得合式，百节各按各职，彼此相助，身体就渐渐增长。也就是说，一个人真正融入教会，不只是“我在这里”，而是“我在这里有我的位置，我在这里承担我的那一份”。不一定是很大的事，可能是接待、关怀、代祷、陪谈、探访、敬拜、儿童、后勤，或者只是忠心地陪伴一个人。重点不是你做得多大，而是你愿不愿意把自己摆上。

弟兄姊妹，寒假已经过去了。今天这个主日，对我们来说，不应该只是节奏恢复正常而已；这应该成为一个重新归回的起点。也许神今天就在呼召我们，从一种松散、漂浮、孤立的属灵状态里出来，重新回到祂的家中，重新回到肢体的连结中。

也许有人会说，我知道教会重要，但我现在状态不好，所以我更不想靠近人。其实很多时候，正因为状态不好，你更需要回到群体里。多马不是等自己先恢复好了才回到门徒当中；恰恰是当他再次与门徒同在时，主向他显现，主恢复了他。

所以，不要因为软弱而远离教会；反而要因为软弱，更加靠近教会。因为神常常不是在我们独自封闭的时候医治我们，而是在群体里、在肢体中、在共同敬拜与交通里重新坚固我们。

当然，我们也要看见更深的一点：我们之所以能够融入教会，不只是靠人的热情，也不是靠一种社交能力，最根本是因为福音。主耶稣基督为我们舍命，使原本与神隔绝的人，得以与神和好；不仅如此，祂也把原本分散的人带进一个新的家里。我们今天能彼此称为弟兄姊妹，不是因为性格相投，不是因为背景相近，而是因为我们同蒙一位主的恩典，同属一个身体。

所以，教会生活的基础不是“我喜欢这些人”，而是“基督先爱了我们，并且把我们放在一起”。这也是为什么，融入教会不是一个可有可无的附加题，而是福音结出的果子。一个真认识基督的人，也会越来越愿意进入基督的身体。

最后，我想用几个很实际的提醒来结束今天的信息。

这个主日之后，第一，不要只停留在“恢复来聚会”，还要立志“恢复与肢体连接”。
 第二，不要只做一个坐在聚会中的人，也要做一个走进关系里的人。
 第三，不要只问“教会能给我什么”，也要问“神今天要我在教会中成为什么样的人”。
 第四，不要因为自己软弱就退后，反而要带着软弱来到主面前，也来到弟兄姊妹中间，让教会成为你重新得力的地方。

愿神借着这个寒假后的第一个主日，再一次把我们从分散中召回，从冷淡中唤醒，从孤立中带回祂的家。愿我们不只是“回到教会”，更是“融入教会”；不只是“人在这里”，更是“心在这里”；不只是“接受恩典”，也愿意“成为别人的祝福”。

阿们。
