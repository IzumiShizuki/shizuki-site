## 1. 最终实现范围（冻结）

一期只交付以下开发面：

1. AI Hub 路由与页面壳层。
2. 普通对话 / 酒馆模式双模式会话。
3. 世界书与角色卡管理。
4. AI 小镇公开漫游与管理员特殊 NPC。
5. RPGMaker 上传 / 解析 / 预览链路。
6. 自宅 companion v1。
7. 管理员专用分层记忆服务。
8. 权限 / 配额 / 守卫整合。

## 2. 不进入一期的实现面（冻结）

1. OpenClaw 接入。
2. Agent 自动做事系统。
3. 多人共享实时地图。
4. RPGMaker 事件运行时。
5. 桌宠驻留态。
6. 多模态音视频交互。

## 5. 实施者默认假设

若无新增产品决策，一律按以下默认值开发：

1. API 路径统一走 `/api/v1/**`。
2. Query 与 JSON 字段统一 `snake_case`。
3. Controller 公共契约使用强类型 DTO。
4. AI 小镇公开浏览不写入记忆。
5. 记忆默认关闭，需管理员主动启用。
6. `Apps` 中的 AI 卡片只作为快捷入口，不作为主实现依赖。
